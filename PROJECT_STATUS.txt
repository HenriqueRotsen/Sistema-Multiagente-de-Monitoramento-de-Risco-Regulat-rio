RESUMO
------------------------------------------------------------------------------

O sistema ja executa um ciclo completo:

1. Coleta publicacoes reais do BCB pela API publica atual.
2. Filtra itens com sinais regulatorios.
3. Analisa documentos com LLM quando configurado.
4. Usa fallback heuristico quando o LLM nao esta disponivel.
5. Gera alertas com resumo, impacto, obrigacoes e recomendacoes.
6. Permite visualizacao basica via Streamlit.
